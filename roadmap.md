# The CoworkerX development Roadmap 
1. initial objective of this agent is to be able to tackle network related issues
2. second objective will be to generate configuration and apply configuration when required
3. the agent should be able to asyncronously receive input from user or from other agents and
spawn a async task  to explore other alternatives injected by user or other agents i.e the agent 
was working on a network issue where users could not connect in a location X then there is a new input 
comes in of a user confirming power outage in few floor, then agent can factor that info and stop
wasting resources exploring network path.
4. agent should tackle network optimization 
5. agent should be capable of reviewing design and even proposing new design ideas to improve the network by just analyzing business objectives.
