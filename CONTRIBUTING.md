# Contributing to WorkerXY

We welcome contributions to WorkerXY.

This project uses the Developer Certificate of Origin (DCO).

By contributing to this repository, you agree to the terms of the DCO
located in the DCO file.

## Signing your commits

All commits must include a Signed-off-by line.

The easiest way to do this is to use:

git commit -s -m "your commit message"

This will automatically add:

Signed-off-by: Your Name [your.email@example.com](mailto:your.email@example.com)

Pull requests without signed commits may be rejected.
